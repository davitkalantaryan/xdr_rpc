# XDR RPC
ONC XDR RPC    
We will use RepoRoot word for repository root directory.  
  

## Compile library  
Open developer command prompt and run there command  
```bat  
msbuild_all.bat  
```  
  
## artifacts location  
'$(RepoRoot)\sys\$(TargetPlatform)\lib' and '$(RepoRoot)\sys\$(TargetPlatform)\dll'  
### target platforms  
 1.  ARM  
 2.  ARM64  
 3.  x64  
 4.  x86
